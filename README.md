# AI-quickdraw-gemini-vision
Google Quick Draw clone powered by Gemini Vision API that recognizes hand-drawn sketches using Generative AI.
An AI-powered drawing game inspired by Google Quick Draw. Users sketch an object on a canvas while Gemini Vision analyzes the drawing in real time and predicts the object using multimodal Generative AI.
